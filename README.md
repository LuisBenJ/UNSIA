# Asistente para Inscripciones de Nuevo Ingreso - UNSIS

Proyecto de tipo prototipo que demuestra la integración de un chatbot en una página web, con la finalidad de apoyar a los aspirantes durante el proceso de inscripción.

El componente principal es un chatbot con información obtenida de fuentes oficiales de Servicios Escolares. Este fue desarrollado con Chatbase e integrado mediante un widget flotante en la página.

---

## ¿Qué hace?

- Muestra información general del proceso de ingreso a licenciatura.
- Permite al aspirante consultar dudas mediante el asistente UNSIA sin salir de la página.

---

## Tecnologías

| Herramienta         | Uso                      |
|---------------------|--------------------------|
| Node.js + Express   | Servidor web            |
| HTML + CSS          | Interfaz de la página   |
| Chatbase            | Motor del chatbot       |

## Instalación

1. **Clona el repositorio**

```bash
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio
```

2. **Instala las dependencias**

```bash
npm install
```

3. **Inicia el servidor**

```bash
node server.js
```

4. **Abre en el navegador**

```
http://localhost:3000
```
